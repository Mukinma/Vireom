# Lucide local icons (required)

Recommended source version: Lucide v0.468.0 (outline style).

Required icons in `raw/`:

- `camera.svg`
- `user.svg`
- `users.svg`
- `shield.svg`
- `lock.svg`
- `unlock.svg`
- `settings.svg`
- `database.svg`
- `check.svg`
- `x.svg`
- `alert-triangle.svg`
- `clock.svg`

This project consumes the local sprite:

- `lucide-sprite.svg`

All symbols are expected to render with `stroke-width="1.5"` and `stroke="currentColor"`.
