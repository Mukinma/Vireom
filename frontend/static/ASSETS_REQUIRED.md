# CameraPI UI local assets

This UI is designed to run fully offline. Do not use CDN imports.

## Fonts

- Family: Inter
- Recommended version: 4.1
- Format: static `.woff2`
- Weights: 400 (Regular), 500 (Medium), 600 (SemiBold)

Store exactly:

- `/static/fonts/inter/Inter-Regular.woff2`
- `/static/fonts/inter/Inter-Medium.woff2`
- `/static/fonts/inter/Inter-SemiBold.woff2`

## Icons

- Set: Lucide Icons (outline)
- Recommended version: 0.468.0
- Stroke width: `1.5`

Raw icon files expected:

- `/static/icons/lucide/raw/camera.svg`
- `/static/icons/lucide/raw/user.svg`
