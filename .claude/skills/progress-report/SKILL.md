# Progress Report

Construye un reporte de avance útil y no cosmético.

## Objetivo
Resumir progreso real para seguimiento, asesoría o defensa.

## Reglas
- No confundas actividad con avance.
- No cuentes como hecho algo no verificado.
- Si hubo retrabajo o bloqueo, dilo.
- Prioriza claridad sobre maquillaje.

## Salida
## Resumen del periodo
## Avances reales
## Evidencia del avance
## Pendientes clave
## Riesgos
## Bloqueos
## Próximo paso
## Mensaje ejecutivo de una sola pieza
