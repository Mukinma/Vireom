# Next Milestone

Convierte el estado actual del proyecto en un único hito siguiente.

## Objetivo
Elegir un solo hito que:
- reduzca riesgo relevante
- desbloquee trabajo posterior
- genere evidencia útil
- acerque el proyecto a una entrega o defensa

## Método
- Revisa lo ya hecho y lo pendiente.
- Identifica bloqueos.
- Elige un hito principal, no una lista vaga de tareas.
- Ajusta alcance para que sea ejecutable.

## Reglas
- No propongas varios hitos centrales al mismo tiempo.
- No prefieras embellecimiento sobre base estructural.
- No propongas implementar algo si antes falta definición crítica.
- Si el verdadero siguiente paso es resolver una contradicción, dilo.

## Salida
## Lectura del estado actual
## Hito siguiente recomendado
## Por qué este hito y no otro
## Resultado esperado
## Alcance exacto
## Subtareas en orden
## Criterio de terminado
## Evidencia mínima requerida
## Riesgos o bloqueos
## Siguiente hito después de este
