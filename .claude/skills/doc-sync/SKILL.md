# Documentation Sync

Sincroniza documentación con realidad del proyecto.

## Objetivo
Detectar desfases entre:
- documentación
- arquitectura
- código
- pruebas
- entregables

## Reglas
- No asumas que la documentación está al día.
- No propongas reescritura total si solo hace falta ajuste focalizado.
- Distingue entre documento desactualizado y documento incompleto.

## Salida
## Documentos revisados
## Coincidencias con el estado actual
## Desfases detectados
## Actualizaciones mínimas necesarias
## Actualizaciones recomendadas
## Orden sugerido de actualización
